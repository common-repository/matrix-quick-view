== Description ==

[Matrix Quick View](https://matrixaddons.com/downloads/quick-view-for-woocommerce/) - WooCommerce Quick View plugin allows you to add a quick view button in the product archive ( loop )  so that your customer can quickly view the product without refreshing or navigating the product description page.

= WooCommerce Quick View - Features =
* Enable/disable quick view
* Custom Color and Background color
* Show/hide on mobile devices
* Modify Quick View button label
* Regular updated
* More feature coming soon

== Installation ==

Install Matrix Quick View for WordPress either via the WordPress plugin directory or by uploading the files to your server at wp-content/plugins.
